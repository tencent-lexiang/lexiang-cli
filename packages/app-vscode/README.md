# 乐享知识库 (lefs)

将乐享知识库以 WebDAV 方式挂载到本地，供 AI IDE 访问。

## 已支持特性

- **多知识库管理**：支持浏览和管理多个乐享知识库。
- **文件浏览**：在侧边栏直接浏览知识库目录结构。
- **文档查看**：支持直接打开查看 `.lxdoc` 文档（只读模式）。
- **WebDAV 挂载**：支持将知识库挂载为本地 WebDAV 驱动器。
- **MCP 集成**：一键复制 MCP 配置片段，方便 AI 助手集成。
- **Chat 集成**：支持将文档添加到 Chat 上下文，辅助 AI 问答。
- **创建文件夹**：支持在知识库中创建新文件夹。
- **日志查看**：提供详细的运行日志，便于排查问题。
